# El problema

Situación:
* Impartir el módulo de sistemas operativos en FP.

Mi definición:
* Debe ser ágil y dinámico de crear => Interpretado + Editor de texto.
* Scripting = comandos del sistema + _"algo que los una de forma lógica"_

Posibles soluciones:
1. Ganar más tiempo
