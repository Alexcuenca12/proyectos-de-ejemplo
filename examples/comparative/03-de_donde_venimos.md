# De dónde venimos

Puesto que los Strings son objetos, debe existir un método constructor definido en la clase que al invocarse crea una instancia del objeto.

* Ejemplo Ruby:

```ruby
>> b = String.new("Bye")
=> "Bye"
>> b.class
=> String
>>
```

```ruby
>> c = String("Good bye")
=> "Good bye"
>> c.class
=> String
```

```python
>>> a = "hola"
>>> a.__len__()
4
>>> len(a)
4
>>>
```

```ruby
>> a = "hello"
=> "hello"
>> a.length
=> 5
>> a.size
=> 5
>>
```
